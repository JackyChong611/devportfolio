Spatial Protein-Expression Prediction from Histology

Trained on ~15 000 spatial-omics spots across four specimens, this pipeline predicts a 38-protein biomarker panel directly from routine biological tissue images. It couples classical ML (RF, SVR) with a fine-tuned PyTorch ResNet50 to turn raw pathology images into quantitative biomarkers. Outputs feed immediately into cost-effectiveness simulations, eliminating the need for additional wet-lab assays.

  • Why it matters for HEOR: The predicted protein panel becomes low-cost, ready-made inputs for budget-impact and CEA models, enabling fast evaluation of biomarker-guided therapies.
  • Generative-AI workflow — Used Claude to scaffold feature-engineering scripts and literature matrices, trimming data-to-prototype time by ~50 %.
  • Result — Classical baseline (PCA + OLS / Random Forest) averaged Spearman ρ = 0.45; fine-tuned ResNet50 lifted this to ρ = 0.72, and flagship biomarker CD11b reached ρ ≈ 0.66—strong enough for scenario analyses.
  • Repro — Python Markdown + GitHub Actions auto-rebuild to meet NICE transparency guidance.

Techniques Used

Pre-proc: RGB → HED, colour norm, patches
Classical ML: RF, SVR, OLS + PCA
Deep Learning: PyTorch ResNet50 TL, GroupKFold
Metrics: RMSE, r, ρ, R²

NHS Dissatisfaction & Political Leanings

Analysed British Social Attitudes surveys (2015–2019, 2021–2023) to quantify how political leaning and demographics shape nine specific reasons for NHS dissatisfaction. Stepwise logistic models isolate ideology-driven vs age-driven effects, controlling for sex, ethnicity, education, employment and post-COVID shocks. Outputs provide probability weights that can feed demand-side modules of health-economic models.

  • Why it matters for HEOR — Dissatisfaction impacts service uptake, adherence and political feasibility of funding scenarios; modelled probabilities can be plugged into cost-effectiveness or budget-impact sensitivity analyses.
  • Key findings — Labour identifiers blame under-funding & staffing; Conservatives cite inefficiency & waste; unaffiliated focus on waiting times. Age 65+ and employment status are independent predictors across all models.
  • Model export — Coefficient tables auto-export to CSV so analysts can drop them into R or Python simulation frameworks (e.g., uptake curves, utility decrements, scenario weights).

Techniques Used

Data wrangling: R tidyverse
Exploratory: ggplot2, Phi & Cramér’s V
Modelling: Stepwise logistic regression
Export: Broom for HE model inputs

Twitter Sentiment Analysis with Natural Language Processing

Built a multi-model sentiment classification system for tweets, using data from SemEval 2017 Task 4A. The project compares traditional machine learning methods and deep learning architectures, incorporating both static and contextual word embeddings to classify tweets as positive, neutral, or negative. The implementation explores how different feature representations (TF-IDF, GloVe, BERT, BERTweet) and preprocessing techniques (e.g. stopword removal) affect performance.

  • Implemented SVM, GaussianNB, MultinomialNB, and LSTM architectures using PyTorch and Scikit-learn.
  • Compared GloVe, BERT, and BERTweet embeddings for word representation.
  • Explored the effect of stopword removal on model accuracy and F1-score.
  • Achieved highest performance with LSTM + BERTweet model (F1 score: 0.774, accuracy: 74.3%).
  • Visualized results with confusion matrices, F1/accuracy plots, and training time comparisons.
